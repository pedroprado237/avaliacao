TAG: Pre

O texto dentro da tag <pre> é exibido em uma fonte de largura fixa e preserva os espaços e as quebras de linha. O texto será exibido exatamente como escrito no código-fonte HTML.
Muito utilizado para exibir trechos de códigos em exemplificações.
Dica: Existem outras tags semelhantes que são capazes de transformar/mostrar os texto em diferentes modelos.
<code> Parte de um código.
<samp> A saida de um código.
<var> Uma variavel no contexto computacional.

Exemplicação:
<pre>
 .body{
  color: red;
 }
</pre>

TAG: Img

A tag <img> é usada para inserir uma imagem em uma página HTML.
Essa tag tem dois atributos principais:
src: especifica o caminho da imagem.
alt: especifica um texto alternativo para a imagem, se por algum motivo a imagem não possa ser exibida.
Nota: A altura e a largura devem ser sempre especificadas. Caso não sejam, a página pode piscar enquanto a imagem é carregada.
Dica: Para que a tag funcione corretamente, você deve sempre especificar o formato da imagem ou gif.

Exemplicação:
<img src="ornitorrinco.png" alt="Erro"
width="500" height="600">

TAG: Video

A tag <video> é usada para incorporar conteúdo de vídeo, como exemplo clipes de filme, dentre outros.
Essa tag contém uma ou mais source tags com diferentes fontes de video. O navegador escolherá a primeira fonte que suporta.
O texto escrito entre esse tag só séra exibido em navegadores que não suportam o elemento video.
Existem 3 formatos suportados em HTML: MP4, WebM e OGG.
Dica: Para arquivos de áudio, de uma olhada para a tag <audio>

Exemplificação:
<video width="320" height="240" controls>
<source src="video.mp4" type="video/mp4">
Seu navegador não suporta vídeos
</video>
